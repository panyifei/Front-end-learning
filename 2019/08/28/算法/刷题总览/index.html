    学习了怎么写 reverse。

    function reverse(nums, a, b) {
    while (a < b) {
    const temp = nums[a];
    nums[a] = nums[b];
    nums[b] = temp;
    a++;
    b–;
    }
    }

    算法总结

    递归：

    1.写下来重复的关系
    2.只要有可能，就存中间结果
    尾递归就是最后执行递归操作，就可以不保存之前函数的调用栈，每次都是更新函数内部的调用栈。优秀啊。尤其是斐波那契数列居然可以这么写。

    function fTail(n, a = 0, b = 1) {
    if (n === 0) return a
    return fTail(n - 1, b, a + b)
    }

    不过所有的递归基本都能通过循环写出来，那种就不需要优化了
